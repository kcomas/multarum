* I Int 1
* F Float 1.0
* C Char "s"
* Y Symbol `something

* L List [a;b;c]
* H Hash [keys]![values]
* F Function {[arg1;arg256]}

* M Module

* / Division, Square Root
* % Remainder
* : Assign
